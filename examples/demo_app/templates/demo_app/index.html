Worker Commands

Run the following commands in another terminal:

# Run once
python manage.py run_redis_tasks

# Continuous mode (poll every 5 seconds)
python manage.py run_redis_tasks --continuous --interval 5

# Run specific queue only
python manage.py run_redis_tasks --queue emails --continuous
